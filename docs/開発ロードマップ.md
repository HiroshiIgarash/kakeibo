### Chapter 1-2: Categoryモデル ― STIと階層構造
**目的：** 変動費・固定費をSTIで表現し、親子関係を持つカテゴリを実装する

学ぶこと：
- STI（Single Table Inheritance）とは何か

- `root?` / `leaf?` の概念
- シードデータの作り方

主なコマンド：
```
rails generate model Category / rails db:seed
```

TDDポイント：
```ruby
# まずテストを書く
it { should belong_to(:parent).optional }
it { should have_many(:children) }
it "子カテゴリを含む合計支出を返す"
```
